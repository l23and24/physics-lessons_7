        📝 Короткий чек-лист для нового уроку:
Якщо ти створюєш новий урок, наприклад 7.80:

Створюєш файл 7.80.html у корені.

Створюєш папку img/L80/.

Закидаєш туди картинки.

У файлі 7.80.html пишеш шлях: src="img/L80/назва_фото.jpg".

Для Google Диску пишемо повну адресу: href="https://drive.google.com/..." або src="https://drive.google.com/..."
