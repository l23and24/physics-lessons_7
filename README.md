Код побудований за принципом Single Page Application (SPA) для головної сторінки, де контент генерується динамічно. Типовий вигляд твого репозиторію в папці має бути таким:

index.html — головний файл . Це «серце» проєкту, де міститься розмітка, стилі та головний скрипт.

7.1.html, 7.2.html, ... 7.105.html — окремі файли для кожного уроку. Скрипт шукає саме їх за назвою.

fizyka-7-klas-bariakhtar-2024.jpg — файл зображення обкладинки підручника.

/img — папка, де зазвичай зберігаються додаткові картинки для уроків.
